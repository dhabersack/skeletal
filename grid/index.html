The grid-mixin allows you to specify a number of columns a container should occupy.

The grid is fully responsive: resize your browser to check how it behaves at different breakpoints. Note that at the smallest breakpoint, all containers expand to span the entire width of the page.

Note: Since Skeletal is based entirely on em-values, you can only safely apply this mixin to elements with a font-size of 1em.

Number of columns

The $number-of-grid-columns-variable in _config.scss influences the layout of the grid and thus the widths the mixin calculates. This variable can be set to 12, 16 or 24 columns.

Floating containers

The $break-every-nth-parameter floats element with the same class and clears the floating after every nth element:

CSS
.item {
  @include grid(
    4,
    $break-every-nth: 3
  );
}

Grid offset

The $offset-attribute allows you to move a container to the right by a number of columns. With additional floating, you can quickly achieve a layout such as the following:

Example

Output

.item-1

.item-2

.item-3

.item-4

CSS
.item-1 {
  @include grid(
    6,
    $offset: 1
  );
}

.item-2 {
  @include grid(
    3,
    $offset: 1
  );
}

.item-3 {
  @include grid(
    4,
    $offset: 2
  );
}

.item-4 {
  @include grid(
    3,
    $offset: 2
  );
}

Centering containers

When the optional $center-parameter is set to true, it is automatically centered within its parent container.

Example

Output

.item

CSS
.item {
  @include grid(
    6,
    $center: true
  );
}

Breakpoint-aware media queries

The grid-mixin allows for quick reorganization of content blocks at different breakpoints through its optional $breakpoints-parameter.

When given a list of breakpoint-names, the corresponding mixin only applies at these breakpoints. Available breakpoint-names are:

  • 'small'
  • 'medium'
  • 'large'
  • 'extra-large'

Example

Output

.item

CSS
.item {
  @include grid(
    10,
    $breakpoints: ('small')
  );

  @include grid(
    9,
    $offset: 1,
    $breakpoints: ('medium', 'large')
  );

  @include grid(
    8,
    $offset: 2,
    $breakpoints: ('extra-large')
  );
}
