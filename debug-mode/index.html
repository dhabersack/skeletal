--- layout: default title: Debug mode ---

You can activate debug mode by setting $is-in-debug-mode in _config.scss to true. This activates both a column-grid background and a baseline-grid based on the value set as $vertical-grid-base-height in the same file:

In order to see the column background, $number-of-grid-columns in _config.scss needs to be set to 12, 16 or 24.
