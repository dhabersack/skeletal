Baseline grid

The font-size-mixin takes an em-based value and sets it as the font-size while also setting the appropriate line-height.

The calculation of the correct line-height takes the variables $vertical-grid-base-height and $minimum-line-height-to-font-size-ratio in _config.scss into account.

The following font-sizes are recommended in order to maintain a solid baseline-grid:

  • 10px
  • 12px
  • 15px
  • 16px
  • 20px
  • 24px
  • 25px
  • 30px
  • 32px
  • 40px
  • 48px
  • 50px
  • 60px
  • 75px
  • 80px
  • 96px
  • 100px
  • 120px

Note: You will have to translate these to em-based values in the individual environments in order for the mixin to calculate the correct results.
