Buttons

Standard

Buttons are created with the background-color of $color-neutral-lighter. They offer subtle highlighting in their :active-state.

Output
HTML
<input type="submit">

Colored buttons

The (optional) $background-color-attribute changes the coloring of a button.

Output
HTML
<input type="submit">
CSS
input[type="submit"] {
  @include button(
    $background-color: $color-highlight
  );
}

Icons

The (optional) $icon-parameter specifies an icon that gets inserted in front of the button text. The following icons are available:

  • $icon-search
  • $icon-check
  • $icon-cross

Note: You can only use the $icon-parameter on button-tags.

Output
HTML
<button id="search">Search</button>
<button id="accept">Accept</button>
<button id="cancel">Cancel</button>
CSS
#search {
  @include button(
    $icon: $icon-search
  );
}

#accept {
  @include button(
    $background-color: $color-action,
    $icon: $icon-check
  );
}

#cancel {
  @include button(
    $background-color: $color-error,
    $icon: $icon-cross
  );
}

Textfields

All form fields are responsive, meaning they resize at set breakpoints.

Standard

The default styling of a text field spans half of the page width at all breakpoints. The color of the border changes to $color-highlight when the field is in focus.

Output
HTML
<input type="text" placeholder="Placeholder">

Widths

The (optional) $number-of-columns-parameter specifies how many columns of the underlying grid a textfield should occupy.

On the smallest breakpoint, textfields always span the full width of the parent container.

Output
CSS
input[type="text"] {
  @include textfield(
    $number-of-columns: 3
  );
}

Breakpoint-aware widths

Identical to the grid-mixin, textfields can be given different widths at different breakpoints through the (optional) $breakpoints-parameter.

Output
CSS
input[type="text"] {
  @include textfield(
    8,
    $breakpoints: ('small')
  );

  @include textfield(
    6,
    $breakpoints: ('medium', 'large')
  );

  @include textfield(
    4,
    $breakpoints: ('extra-large')
  );
}
