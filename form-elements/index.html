Buttons

Standard

Buttons are created with the background-color of $color-neutral-lighter. They offer subtle highlighting in their :active-state.

HTML
<input type="submit">

Colored buttons

You can specify an optional $background-color-attribute to change the color of the button. Setting this parameter also changes the border color.

HTML
<input type="submit">
CSS
input[type="submit"] {
  @include button(
    $background-color: $color-highlight
  );
}

Icons

With the optional $icon-parameter, you can specify an icon to insert in front of the button text.

Currently, the following icons are supported:

  • $icon-search
  • $icon-check
  • $icon-cross

Note: You can only use the $icon-parameter on button-tags.

Search-button
HTML
<button>Search</button>
CSS
button {
  @include button(
    $icon: $icon-search
  );
}
Accept-button
HTML
<button>Accept</button>
CSS
button {
  @include button(
    $background-color: $color-action,
    $icon: $icon-check
  );
}
Cancel-button
HTML
<button>Cancel</button>
CSS
button {
  @include button(
    $background-color: $color-error,
    $icon: $icon-cross
  );
}
