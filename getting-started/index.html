--- layout: default title: Getting started ---

Skeletal assumes you are running Sass. If it is not yet installed on your machine you can get it by running: 

gem install sass

Once the installation is complete you can grab the source on GitHub and drop the skeletal-directory in with your stylesheets.

Add the following line at the top of your Sass-stylesheet to get access to all variables and mixins in your own styles: 

@import 'skeletal/skeletal.scss';
