--- layout: default title: Getting started ---

Skeletal requires Sass, which you can install by running: 

gem install sass

Once the installation is complete you can grab the source on GitHub and drop the skeletal-directory in with your stylesheets.

Add the following line at the top of your Sass-stylesheet to gain access to all variables and mixins in your own styles: 

@import 'skeletal/skeletal.scss';

No further configuration is required as long as you continue compiling your stylesheet with sass watch as usual.
