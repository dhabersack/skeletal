Skeletal

A clean foundation for a lovely frontend.

Skeletal is a framework that provides a collection of default styles and Sass-mixins that help getting up and running on webdesign projects a lot quicker than writing everything from scratch does.

  3. Tried and tested

    Skeletal grew out of real-world projects — its contents are robust and can be used to create a wide range of layouts.

  4. Clutter-free

    Keep your markup clean: all presentational attributes are kept in stylesheets for easy maintenance.

  5. Adaptable

    Change the defaults and select which features you want to include to make the system meet your requirements.

Documentation

Skeletal is built on Sass. Besides setting sensible defaults for the styling of most tags, it offers a lot of functionality through variables and mixins.

Colors

_colors.scss contains a list of colors you want to use in your projects with the defaults being: 

$color-blue:   #3581fc;
$color-gray:   #bdbdbd;
$color-green:  #23db2a;
$color-red:    #e33d27;
$color-violet: #904fff;
$color-yellow: #ffe226;

You then create named proxies for them in _config.scss: 

$color-action:       $color-green;
$color-alert:        $color-yellow;
$color-brand:        $color-yellow;
$color-debug:        rgba($color-red, 0.0625);
$color-error:        $color-red;
$color-highlight:    $color-yellow;
$color-link:         $color-blue;
$color-link-visited: $color-violet;
$color-neutral:      $color-gray;

For all but $color-debug, Skeletal generates lighter and darker versions in _setup.scss by creating additional variables. This puts the following default palette at your disposal:

Floating containers

The $break-every-nth-parameter floats element with the same class and clears the floating after every nth element: 

.item {
  @include grid(4, $break-every-nth: 3);
}

Grid offset

The $offset-attribute allows you to move a container to the left by a number of columns. With additional floating, you can easily achieve a layout such as the following:

grid(6, $offset: 1)

grid(3, $offset: 1)

grid(4, $offset: 2)

grid(3, $offset: 2)

Number of columns

You can set the $number-of-grid-columns-variable in _config.scss to 12, 16 or 24.

Changes to this variable have a direct effect on the widths the grid-mixin calculates.
