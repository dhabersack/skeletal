Introduction

Skeletal is a reset preset-stylesheet that sets sensible defaults for most (for now) HTML5-tags, providing a solid base to build upon.

Because a reset just isn't good enough.

Clean
It does not clutter your markup with cryptic classes and IDs by using expressive, semantic tags. This allows for a very clean markup.
Easily configurable
Change the utilized colors and fonts, adapt parts of the layout by adjusting the configuration found in _config.scss.
Pretty
Look at all the nice colors! Seriously tough, Skeletal puts an emphasis on readability and clean organization to make the sites using it as easy and user-friendly as possible.

Status

Skeletal is a work in progress. Because of that, not all tags are completely styled yet. Below you can find a list of all finised and unfinished tags.

Styled tags

abbr, article, b, blockquote, body, code, dd, dl, dt, em, h2, h3, h4, h5, h6, i, ins, pre, q, strong

Partially styled tags

a, address, aside, button, cite, del, dfn, fieldset, footer, form, h1, header, hr, html, input, kbd, label, legend, li, mark, nav, ol, optgroup, option, p, samp, section, select, table, tbody, td, textarea, tfoot, th, thead, tr, ul, var

Unstyled tags

area, audio, bdi, bdo, br, canvas, caption, col, colgroup, datalist, details, div, embed, figcaption, figure, hgroup, iframe, img, keygen, map, menu, meter, object, output, param, progress, rp, rt, ruby, s, small, span, sub, summary, sup, time, u, video, wbr
