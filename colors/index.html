Skeletal promotes the use of proxy-names for variables by mapping color-based names to purpose-based names.

_colors.scss contains the colors used in your projects: 

$color-blue:   #3581fc;
$color-gray:   #bdbdbd;
$color-green:  #23db2a;
$color-red:    #e33d27;
$color-violet: #904fff;
$color-yellow: #ffe226;

The variables you are going to use are set in _config.scss: 

$color-action:       $color-green;
$color-alert:        $color-yellow;
$color-brand:        $color-yellow;
$color-debug:        rgba($color-red, 0.0625);
$color-error:        $color-red;
$color-highlight:    $color-yellow;
$color-link:         $color-blue;
$color-link-visited: $color-violet;
$color-neutral:      $color-gray;

For all but $color-debug, Skeletal generates lighter and darker versions in _setup.scss. Note that this only works for this exact set of variables.
