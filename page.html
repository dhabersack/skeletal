Skeletal

The web does not need another frontend-framework. Yet here we are.

Skeletal is a frontend-framework that works really well for me. It is a collection of default stylings and SASS-mixins that help me get up and running on webdesign projects a lot quicker than writing everything from scratch does.

What is in here

  1. Readable

    Clean organization and default typography settings keep the sites simple and user-friendly.

  2. Responsive

    The Sass-powered grid grows with the browser to adapt to a variety of devices.

  3. Semantic

    No need to clutter markup with classes and IDs to create a layout: it is all done in stylesheets, the way it should be.

  4. Adaptable

    Defaults for things like fonts, colors, grids and more can be configured and extended to meet custom requirements.
